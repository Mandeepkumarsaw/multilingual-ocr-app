# 🧠 Multilingual OCR App

A powerful OCR (Optical Character Recognition) web application that extracts text from images and PDFs using Tesseract OCR and Streamlit.

---

## 🚀 Features

- 📷 Image to Text Extraction
- 📄 PDF to Text (Multi-page support)
- 🌐 Multilingual OCR (Indian Languages Supported)
- ⚙️ Image Preprocessing (Grayscale, Noise Removal, Thresholding)
- 🧠 Supports Hindi, English, Tamil, Telugu, Bengali, and more
- ⬇️ Download extracted text

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Tesseract OCR
- OpenCV
- PyMuPDF (fitz)
- PIL (Pillow)
